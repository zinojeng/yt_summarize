# 專案進度追蹤

本文件記錄 YouTube 影片摘要生成器的開發進度、待辦事項和未來規劃。

## 已完成功能

- [x] 使用 yt-dlp 下載 YouTube 影片
- [x] 將影片轉換為音訊格式 (MP3)
- [x] 自動分割超過大小限制的音訊檔案
- [x] 使用 GPT-4o-transcribe 進行音訊轉文字
- [x] 支援 Google Gemini 和 OpenAI 的多模型架構
- [x] 改進摘要提示，生成更具分析性的摘要
- [x] 檔案和目錄自動管理
- [x] 完整的錯誤處理和日誌記錄
- [x] 命令列參數支援 (--keep-audio, --log-level)
- [x] 基本的 FastAPI Web 介面框架
